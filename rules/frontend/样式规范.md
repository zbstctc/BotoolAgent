# 前端样式规范

## 设计理念：岩石色系 (Rock Palette)
- **整体风格**：黑白灰为主的单色系，沉稳、专业、克制
- **按钮全部黑色**：主按钮用黑底白字，不使用蓝色/紫色等彩色按钮
- **彩色仅用于状态指示**：成功(绿)、错误(红)、警告(琥珀)、进行中(蓝) — 只用在小元素（徽章、圆点、状态灯），不用在按钮上
- **禁用彩色按钮**：`bg-blue-600`、`bg-purple-600`、`bg-indigo-600` 等彩色按钮一律禁止

## Tailwind CSS 优先
- 所有样式必须使用 Tailwind 工具类，禁止自定义 CSS（除 `globals.css` 中的动画定义）
- 不使用 CSS Modules 或 styled-components

## 调色板

### 主色系：neutral（唯一主色系）
| 用途 | 色值 | Tailwind 类 |
|------|------|-------------|
| 纯黑（按钮、标题） | `#171717` | `neutral-900` |
| 深灰（正文） | `#525252` | `neutral-600` |
| 中灰（辅助文字） | `#737373` | `neutral-500` |
| 浅灰（提示、placeholder） | `#a3a3a3` | `neutral-400` |
| 边框 | `#e5e5e5` | `neutral-200` |
| 浅底色 | `#f5f5f5` | `neutral-100` |
| 背景 | `#fafafa` | `neutral-50` |

### 文本层级
- 一级标题/重要内容：`text-neutral-900`
- 正文：`text-neutral-600`
- 辅助说明：`text-neutral-500`
- 提示/placeholder：`text-neutral-400`

### 状态色（仅用于小元素，不用于按钮）
| 状态 | 用途 | 徽章/标签 | 圆点/状态灯 | 背景提示框 |
|------|------|-----------|-------------|------------|
| 成功 | 完成、通过 | `bg-green-100 text-green-700` | `bg-green-500` | `bg-green-50 border-green-200 text-green-700` |
| 错误 | 失败、异常 | `bg-red-100 text-red-700` | `bg-red-500` | `bg-red-50 border-red-200 text-red-600` |
| 警告 | 注意 | `bg-amber-100 text-amber-700` | `bg-amber-500` | `bg-amber-50 border-amber-200 text-amber-700` |
| 进行中 | 活跃、加载 | `bg-neutral-200 text-neutral-700` | `bg-neutral-500` | `bg-neutral-100 border-neutral-200 text-neutral-600` |

> **注意**：`bg-blue-100 text-blue-700` 等蓝色标签应迁移为 `bg-neutral-200 text-neutral-700`

## 按钮样式

### 主按钮（黑底白字）— 唯一的 CTA 按钮
```
bg-neutral-900 text-white hover:bg-neutral-800
disabled:bg-neutral-300 disabled:text-neutral-500 disabled:cursor-not-allowed
```
适用：保存、提交、确认、开始、创建、下一步

### 次按钮（白底黑框）
```
border border-neutral-300 text-neutral-700 bg-white
hover:bg-neutral-50 hover:border-neutral-400
disabled:bg-neutral-100 disabled:text-neutral-400 disabled:cursor-not-allowed
```
适用：取消、返回、辅助操作

### 危险按钮（红色 — 唯一允许的彩色按钮）
```
bg-red-600 text-white hover:bg-red-700
disabled:bg-red-300 disabled:cursor-not-allowed
```
适用：删除、停止、终止（需要用户特别注意的破坏性操作）

### 幽灵按钮（无边框）
```
text-neutral-600 hover:text-neutral-900 hover:bg-neutral-100
```
适用：关闭、折叠、次要链接

### 统一基础类
```
px-4 py-2 rounded-lg font-medium text-sm transition-colors
```

### 禁止使用的按钮配色
- ~~`bg-blue-600 text-white`~~ → 改用 `bg-neutral-900 text-white`
- ~~`bg-purple-600 text-white`~~ → 改用 `bg-neutral-900 text-white`
- ~~`bg-indigo-600 text-white`~~ → 改用 `bg-neutral-900 text-white`
- ~~`bg-green-600 text-white`~~ (作为按钮时) → 改用 `bg-neutral-900 text-white`

## 圆角与阴影
- 常规容器：`rounded-lg`
- 小元素（badge、tag）：`rounded-md`
- 圆形（头像、状态灯）：`rounded-full`
- 卡片阴影：`shadow-sm`（普通）、`shadow-xl`（弹窗/Modal）

## 边框
- 默认边框：`border border-neutral-200`
- 强调边框：`border-2 border-neutral-900`
- 激活状态：`ring-2 ring-neutral-200 border-neutral-400`

## 交互状态
- Hover：`hover:bg-neutral-50`、`hover:border-neutral-300`、`hover:shadow-sm`
- Focus：`focus:border-neutral-900 focus:ring-2 focus:ring-neutral-200 outline-none`
- Disabled：`disabled:bg-neutral-200 disabled:text-neutral-400 disabled:cursor-not-allowed`
- Active/pressed：`active:bg-neutral-800`

## 布局
- 使用 Flex 布局为主（`flex items-center justify-between`）
- 容器宽度约束：`max-w-7xl mx-auto`
- 间距使用 `gap-2`、`gap-3`、`gap-4`
- 内边距：`p-4`、`px-6 py-4`

## 弹窗/对话框（重要）
- 所有弹窗和下拉框必须使用白色背景（`bg-white`），严禁透明
- 遮罩层：`bg-black/50`
- 弹窗容器：`bg-white rounded-lg shadow-xl`

## 图标
- 使用内联 SVG，不使用图标库
- 标准尺寸：`w-4 h-4`（小）、`w-5 h-5`（中）、`w-6 h-6`（大）
- 颜色继承：`stroke="currentColor"` 或 `fill="currentColor"`
- 线宽统一：`strokeWidth={2}`

## 过渡动画
- 颜色变化：`transition-colors`
- 所有属性：`transition-all`
- 自定义动画定义在 `globals.css`，类名前缀 `task-card-`
