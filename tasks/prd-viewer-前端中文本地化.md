# PRD: Viewer 前端中文本地化

**文件位置:** `tasks/prd-viewer-chinese-localization.md`

## 需求摘要

| 维度 | 决策 |
|------|------|
| **范围** | Viewer 全部页面 |
| **目的** | 面向国内用户 |
| **方式** | 直接替换硬编码文本 |
| **文本类型** | 只翻译界面文本（按钮、标题、导航等） |
| **术语处理** | 保留英文术语（PRD、API、Stage 等） |
| **翻译风格** | 简洁口语化 |
| **按钮格式** | 动词形式（开始、提交、取消） |
| **状态格式** | 完整句子（正在加载...、操作已完成） |
| **动态内容** | 只翻译静态文本 |
| **排除范围** | 变量名、日志、注释都不翻译 |
| **验收方式** | 浏览器中查看 |
| **多语言** | 不需要预留 |
